how it works

step 1: a bit of JavaScript


function gotAssertion(assertion) {
  $.post('/login',
         {assertion:assertion,
          csrf:csrf},
         function(result) {
           // update display
         });
}

navigator.id.get(gotAssertion);

step 2: verify on the server


$ curl -d "assertion=<ASSERTION>
          &audience=https://mysite.com"
          "https://browserid.org/verify"
{
    "status": "okay",
    "email": "lloyd@example.com",
    "audience": "https://mysite.com",
    "expires": 1308859352261,
    "issuer": "browserid.org"
}

Did you notice...


  • no API keys
  • clear disclosure: just an email address
  • works for everyone, not just users of X
  • 2-click login. The 2nd click is user consent.
